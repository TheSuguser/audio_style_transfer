# Audio Style Transfer

This is a TensorFlow implementation of audio style transfer algorithm.

## Dependencies

* Python (tested with 3.5)

* TensorFlow

* numpy

* matplotlib

* LibROSA

  ~~~
  pip install librosa
  ~~~

### How to run

* Open ```train.py```
* Set ```content_name``` , ```style_name``` and ```ALPHA```
* Run ```train.py```
